#  Recetario de Cócteles Hecho por Joan Marqués (joanmarquésverdegal1@gmail.com)


Una aplicación web interactiva para consultar y aprender recetas de cócteles.  
Diseñada para uso interno en una coctelería

---

##  Características

- **Interfaz** con imágenes de cada cóctel.  
- **Buscador** para encontrar rápidamente una receta.  
- **secciones**: carta, coctelería clásica, cócteles de autor, etc.
